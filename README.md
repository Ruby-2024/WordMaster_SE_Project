# 📚 WordMaster - 单词大师软件项目

## 🌟 项目简介 (Introduction)

本项目是《软件工程》课程设计项目，旨在开发一款基于网页（Web）的背单词应用。
在实现核心背记功能的基础上，我们创新性地引入**深度游戏化（Gamification）机制**，以解决传统背单词软件枯燥、用户粘性低的问题，显著提高用户的学习动力和长期坚持度。

**项目特色：**

1.  **官阶晋升机制：** 根据用户掌握的词汇量、难度系数和学习频率，计算并授予荣誉“官阶”，提供持续的成就感和学习动力。
2.  **实时排位赛（PVP）：** 允许用户匹配对手进行限时单词抢答比赛，通过竞技升星、掉星，打造竞争和社交的学习环境。

**当前状态：** 已完成初步网页原型开发，正在进行需求细化与核心架构设计。

## 💻 软件架构 (Software Architecture)

### 模块划分（基于思维导图）

本项目采用前后端分离的架构，主要分为以下核心模块：

| 模块名称 | 职责描述 | 主要负责人 |
| :--- | :--- | :--- |
| **核心背记模块** | 负责单词展示、复习算法（记忆曲线）、学习进度统计。 | 核心开发/后端 |
| **游戏化模块** | 负责官阶计算逻辑、排位赛匹配算法、排行榜数据管理。 | 核心开发/后端 |
| **用户与内容管理** | 负责用户注册/登录、词库数据的增删改查、生词本管理。 | 核心开发/后端 |
| **前端/用户界面** | 负责 UI/UX 设计、所有页面的交互实现、数据展示。 | 前端/UI |
| **项目管理与质量** | 负责项目进度、文档管理、版本控制和测试。 | 队长/质量保障 |

## 📝 使用说明 (Usage Instructions)

**(此部分将在核心功能实现后进行填充)**

* **初次使用：** 注册账号，选择一本词书。
* **开始学习：** 点击“开始背单词”，系统将根据艾宾浩斯曲线推送新词或需要复习的词汇。
* **官阶晋升：** 坚持学习并掌握高难度词汇，词汇水平达到要求即可自动晋升官阶。
* **参与排位赛：** 在“竞技场”模块，点击“匹配对手”进行实时对战。
